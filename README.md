---

## Overview
This repository contains the essential resources to understand the **LLM-based workflow** used to design, validate, and prototype a high‑frequency resonant **wireless power transfer (WPT)** system. It includes:

- **`pseudocode.md`** – Algorithmic workflow matching the paper’s figure (two validation stages: analytical → simulation → prototyping).
- **`llm_instructions.md`** – The analytical equations and rules supplied to the LLM agent.
- **`LICENSE`** – CC BY 4.0 license for academic reuse with attribution.

> The repository may remain private during peer review and will be made public upon publication.
